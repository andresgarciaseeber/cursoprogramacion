SOLUCIÓN DE PROBLEMAS:

- Error 404 en rutas:
  Verifica que mod_rewrite esté activo y .htaccess esté presente

- API no responde:
  Revisa la URL en la configuración del build (.env.production)
  Asegúrate que CORS esté habilitado en api.php

- Página en blanco:
  Abre la consola del navegador (F12) para ver errores
  Verifica que los assets se carguen correctamente
